# simple_clicker
Simple clickers writen on autoit

## shadow_priest.au3

ДПСит shadow priest в world of warcraft 3.3.5. Плюшки:
- регенит ману
- держит все доты, ДПС около 5.5к на маникене в огри под своими бафами
- прожимает берсерк по кд

## buggy_walk.au3

Позволяет передвигаться вперёд мелкими шажками, что очень важно пользующимся speedhack'ами
